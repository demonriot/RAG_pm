# Data Sources

The data extracted and used in this project originates from three scraped HTML files. These files were fetched from the Oregon State University (OSU) Academic Catalog and are stored locally in the `tmp/` directory.

### 1. `tmp/tet.html`
*   **Title:** Course Descriptions < Oregon State University
*   **Description:** The main index page for OSU course descriptions. It lists all the different academic subjects, colleges, and departments with links to their respective detailed course listings.
*   **Original URL:** [https://catalog.oregonstate.edu/courses/](https://catalog.oregonstate.edu/courses/)

### 2. `tmp/tet2.html`
*   **Title:** Computer Science (CS) < Oregon State University
*   **Description:** The complete list of course descriptions for the Computer Science (CS) department at OSU. It contains course codes, titles, credits, descriptions, and prerequisite information for undergraduate and graduate CS courses.
*   **Original URL:** [https://catalog.oregonstate.edu/courses/cs/](https://catalog.oregonstate.edu/courses/cs/)

### 3. `tmp/tet3.html`
*   **Title:** Academic Regulations < Oregon State University
*   **Description:** The academic policies and regulations page for OSU. It details institutional rules on admissions, transfer credits, course registration, grading systems, academic standing (probation/suspension), and graduation requirements.
*   **Original URL:** [https://catalog.oregonstate.edu/regulations/](https://catalog.oregonstate.edu/regulations/)
